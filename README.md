博客分类管理操作开发纪要
============
# 专题菜单的开发概要
1. 专题菜单的本质是博客分类，在博客分类管理中进行管理，同时也能够在菜单管理中进行专题菜单的管理
2. 专题菜单的创建流程如下：
    1. 初始状态下专题菜单的数目为空，用户选择点击创建专题博客
    2. 输入必要信息创建专题博客-->自动创建一个博客分类（博客分类中有扩展字段可以使用）
    3. 创建专题完成后
    
3. 专题菜单的层级调整操作：
    1. 调整专题菜单，不对对应的博客分类进行处理，但：
        1. 从叶子节点调整到非叶子节点：将原叶子节点中对应博客分类的所有博客转移到默认分类
        2. 从非叶子节点调整到叶子节点
    2. 专题菜单的核心：非叶子节点下不允许有博文
        1. 删除一个专题菜单，如果该菜单是叶子节点，判断当前菜单下是否有博文，如果有提醒用户操作
        2. 如果删除一个非叶子节点的专题菜单，判断该菜单下的所有叶子节点是否悬挂了博文，如果有提醒用户操作
